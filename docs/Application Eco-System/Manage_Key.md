# Manage Key

## Overview

You can download your key from here.

You can see all key related information.

You can deactivate your unused instances.

**Options:**

1.   Download

1.   C-Code

1.   Key Name

1.   Generated on

1.   CPU Usage

1.   Concurrent Connections

1.   Concurrent Users

1.   Named Users

1.   Renew Support

1.   Instance generated

1.   Instance
