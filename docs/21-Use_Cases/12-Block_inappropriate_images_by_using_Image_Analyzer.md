# Block inappropriate images in real time with Image Analyzer

Use the Image Analyzer to prevent users from viewing pornographic or inappropriate images by analyzing graphical content in real time.



## Prerequisites
HTTPS Inspection should be enabled in SafeSquid. If not enabled, see the document - [How to enable HTTPS Inspection](/docs/SSL_Inspection/main/).
